Welcome to XAMPP for Windows 7.4.29

You have successfully installed XAMPP on this system! Now you can start using Apache, MariaDB, PHP and other components. You can find more info in the FAQs section or check the HOW-TO Guides for
getting started with PHP applications.

XAMPP is meant only for development purposes. It has certain configuration settings that make it easy to develop locally but that are insecure if you want your installation accessible to others. If you
have want your XAMPP accessible from the Internet, make sure you understand the implications and check the FAQs to learn how to protect your site. Alternatively, you can use WAMP, MAMP, or LAMP which
are similar packages more suitable for production.

Start the XAMPP Control Panel to check the server status.
